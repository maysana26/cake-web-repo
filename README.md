🍰 Cakes Website (Full-Stack E-Commerce Project)
📌 Overview

This is a full-stack e-commerce website designed for a cake and sweets shop.
It provides users with a smooth shopping experience, modern UI design, and a functional backend for managing products and orders.

The project was built as part of an academic/portfolio project to demonstrate frontend, backend, and database integration skills.

✨ Features
🛍️ Product browsing (cakes, sweets, desserts)
🧁 Clean and modern UI design
🔍 Product details page
🛒 Shopping cart functionality
👤 User-friendly navigation
🗄️ Database integration (MySQL)
🔐 Basic authentication system (if included)
📱 Responsive design (mobile & desktop)
🛠️ Technologies Used
HTML5
CSS3
JavaScript
PHP
MySQL
XAMPP / local server environment

🚀 How to Run the Project Locally
Install XAMPP or any local server (Apache + MySQL)

Move the project folder into:

htdocs
Start Apache and MySQL from XAMPP
Import the database:
Open phpMyAdmin
Create a new database
Import database.sql

Open your browser and go to:

http://localhost/cakes-website
